Frozen F&B Project Overview

This is a project which allow to use a real world data to generate a visualisation graph. Firstly I use microsoft excel do to data cleaning and formatting so that the data was neat and tidy. Due to some format is not recognise by the Big Query so I have use other tools which is Microsoft SQL Server to change the format.

After the formatting of the raw data. The data that is clean with different part will be merge using Google Cloud Platform Big Query. After merging the whole file, I continue by using tableau to generate a clear and resonable visualisation graph. Lastly I have make a presentation slides by using powerpoint.

Tools Used

Microsoft Excel
GCP(Big Query)
Tableau
Microsoft PowerPoints
Microsoft SQL Server
